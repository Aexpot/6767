## Дополнительно: Раздел "Доступ" в админке

Добавлен новый раздел в админ-панели для управления IP-адресами:

### Функционал:
- Добавление IP-адресов в белый список
- IP из белого списка могут открывать приложение без Telegram
- Удобно для тестирования и офисных IP
- Описание для каждого IP
- История добавления (кто и когда добавил)

### Использование:
1. Откройте админ-панель
2. Перейдите на вкладку "Доступ"
3. Введите IP-адрес (например, 178.66.85.0)
4. Добавьте описание (необязательно)
5. Нажмите "Добавить IP"

Теперь пользователи с этого IP смогут открывать приложение напрямую, без проверки Telegram.
